1.1 在Django下配置Mysql数据库

首先进入数据库的命令行环境下

 ##创建数据库
 create database findcause;

如图所示提示安装成功

回到Django中按照如下格式配置数据库

DATABASES = {
    'default': {
        'ENGINE': 'django.db.backends.mysql',
        'HOST': DB_HOST,
        'PORT': DB_PORT,
        'NAME': DB_NAME,
        'USER': DB_USER,
        'PASSWORD': DB_PASSWORD,
        'OPTIONS': {'charset': 'utf8mb4', 'use_unicode': True},
    }
}
