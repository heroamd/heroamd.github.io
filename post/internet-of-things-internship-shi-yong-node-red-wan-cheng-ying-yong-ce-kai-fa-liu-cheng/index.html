2021-07-07 6 min read # IoT

一、Node-RED背景介绍

· Node-Red是IBM公司开发的一个可视化的编程工具。它允许程序员通过组合各部件来编写应用程序。这些部件可以是硬件设备(如：Arduino板子)、Web API(如：WebSocket in和WebSocket out)、功能函数(如：range)或者在线服务(如：email)。

· Node-Red提供基于网页的编程环境。通过拖拽已定义node到工作区并用线连接node创建数据流来实现编程。程序员通过点击‘Deploy’按钮实现一键保存并执行。程序以JSON字符串的格式保存，方便用户分享、修改。

· Node-Red基于Node.js，它的执行模型和Node.js一样，也是事件驱动非阻塞的。理论上，Node.js的所有模块都可以被封装成Node-Red的一个或几个node。（Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。使用了一个事件驱动、非阻塞式 I/O 的模型，使其轻量又高效。 Node.js 的包管理器 npm是全球最大的开源库生态系统。）

三、Node-RED总结

1、简介

IBM公司开发，可视化编程工具，可以应用于各种硬件设备，基于网页编程环境，基于Node.js（npm是包管理器，全球最大的生态系统）。

Node-RED 连接输入节点和输出节点是采用数据流的方式，部署按键有两个功能，一、保存；二、部署。在添加节点之后节点的右上角有一个蓝色的圆点，这个圆点的意思是，此节点还没有部署和保存。

2、调试

“inject”节点可以手动输入消息，节点左侧有一个小按钮，点击按钮可以手动注入消息。

在点击inject节点的按钮之前，必须确保debug节点是可用的，即按钮必须是“伸出来”的，而不是像按钮3一样“缩回去”，按钮“缩回去”的debug节点不工作。点击按钮可以切换节点是否工作。

四、常用控件介绍

1、inject注入，输入时间戳

时间在线计算器：入口

如需输入数组，在“内容”一栏，要选择二进制，并且手动添加英文字符中的括号与逗号。

总结：inject需要了解的参数有：时间戳、内容和主题。其中内容主要用到数字和二进制流两类，二进制流指的是数组。内容可以为十进制也可以为十六进制，还可以输入ASCII码，利用inject的二进制流可以进行ASCII码与字符的转换（需要在缓冲区编辑器中查看），如果需要输入回车换行等无法在文字列中中输出的字符必须用ASCII码来表示。

2、主题与名称

总结：节点属性中也可以配置主题（topic）与名称（name）。name属性只影响了工作区中节点的外观，并没有影响流中数据包的内容。
Topic是数据包本身带的一个属性，其信息会随着数据包下发。在后续的学习中，经常会用到topic来判断数据包的来源。

3、重复发送

使用间隔和特定时间间隔两种， "特定时间内间隔" 和 "特定时间" 会使用cron系统。区别在于cron系统的时间只看“整”的时间，或者说“标准的”时间，并不关心程序是什么时候开始运行的。

4、Debug控件介绍

调试窗口自带有过滤功能，可以过滤掉不需要的消息。

5、Switch控件

Switch本意是开关、转换，用在node-red里有点像一个岔路口，node-red里边有数据流，数据流可能有不同的分支，在“岔路口”管理数据流向的。Switch控件可以根据数据的topic或是payload，或者其它属性来判断数据该流向哪一个出口。

6、函数控件实现多个输出

从严谨的角度来讲，函数控件中的return不应该称为“返回值”，应为它没有返回任何一个东西给“调用”它的节点。它真正的作用，是传递参数给下一个节点。所谓的返回值，只是一个习惯性的说法而已。

7、通过判断语句进行数据分类

输入与输出不变，把函数节点替换掉switch节点，并修改代码：

if (msg.payload >10 ){
    return [msg,null,null];
}else if(msg.payload === 10 ){
    return [null, msg,null];
}else{
    return[null,null,msg];   //分别代表第一路，第二路，第三路输出
}

注意：三个等号我们称为等同符,当等号两边的值为相同类型的时候,直接比较等号两边的值,值相同则返回true,若等号两边的值类型不同时直接返回false。

8、单个数字的计算

用函数节点对数字进行计算 : var newMsg = {payload: msg.payload * 2 }语句的意思是，新建一个对象，名为newMsg，它有payload属性，并且payload的值为msg.payload的两倍。最后返回的是newMsg，也就是新建的这个对象。

注意：return的返回值时下一个节点的payload

如果不希望新建一个对象，直接修改msg.payload的方法：

msg.payload = msg.payload*2;
return msg;
