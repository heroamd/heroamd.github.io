Daily: Replace with matual blog web construction

2020-03-12 1 min read # Blog

​ 今天花了半个小时时间将原来的yilia主题更换为了Ayer主题。根据作者的描述，Ayer一词词出多处，其中马来语中是“水”的意思，而在西班牙中是“昨天”的意思。可能是发音引起了所有种族人群的共鸣，这个词再两个截然不同的文明中竟然都带有一点点忧郁，纯净的气质。主题如其名，淡淡的，非浓郁的，清新的，轻，的感觉，第一眼就它了。

分类

hexo new page categories

然后将以下复制到/source/categories/index.md文件

---
title: categories
type: "categories"
layout: "categories"
---

标签

hexo new page tags

配置同分类一样

相册

需要在文章顶部配置

---
title: Gallery

albums: [
        ["img_url","img_caption"],
        ["img_url","img_caption"]
        ]
---
