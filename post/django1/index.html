1.7 配置git远端服务器（测试）

##本地部分

git init //托管目录
git add . // 将所有目录做一个提交
git commit -m 'text1' //做一个提交的记录

##将本地仓库同步到远端

git push https://github.com/heroamd/django.git master

##为了后续更加方便的推送到远端，给仓库地址添加一个别名

git add origin https://github.com/heroamd/django.git //origin就代表后面仓库的地址
    
##此后需要继续推送运行

git push origin master
